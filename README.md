# YungDrip Web Platform

Custom e-commerce platform built with Next.js replacing Shopify.
This project includes:

- User storefront
- Admin dashboard
- Product management system
- Shipping integration
- StyleMe (3D styling engine) – separate module

This repository will contain the core web platform.
StyleMe will be developed as a separate service.

---

## 🚀 Tech Stack

Frontend:
- Next.js
- TailwindCSS
- Zustand / Redux

Backend:
- Node.js / Next API
- MongoDB / PostgreSQL

Storage:
- Cloudinary / AWS S3

Auth:
- JWT / Clerk / Firebase

Shipping:
- Shiprocket API / Delhivery API

3D Engine (StyleMe only):
- Three.js
- React Three Fiber

---
